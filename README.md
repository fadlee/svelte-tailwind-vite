# Svelte + Tailwind + Vite

Get up and running with Svelte, Tailwind and Vite.

## Installation

```
npx degit "fadlee/svelte-tailwind-vite#main" my_app
cd my_app
npm install
```

## Scripts

### Start a development server

`npm run dev`

### Build the application

`npm run build`

### Serve the application

`npm run serve`
